# 从这里开始

Repo as Review OS 是一个基于 GitHub 和人工智能协作的培训与复习系统。

它的目标是让人类用户和 AI 工具一起维护一个结构化复习仓库。

## 复制给 AI 的第一句话

```text
请阅读这个仓库里的 START-HERE.zh-CN.md、README.zh-CN.md 和 AGENTS.zh-CN.md。帮我搭建自己的 Review OS。先解释它是什么，再指导我创建或选择一个 GitHub 仓库，然后创建基础文档、AI 接手规则、模板和网站文案。
```

## AI 应该怎么做

1. 先读本文件。
2. 再读 `README.zh-CN.md`。
3. 再读 `AGENTS.zh-CN.md`。
4. 向用户解释这个系统。
5. 指导用户创建或选择一个仓库。
6. 搭建目录结构。
7. 添加模板和接手规则。
8. 汇报创建或修改了哪些文件。

## 人类应该怎么做

1. 创建一个 GitHub 仓库。
2. 把 AI 工具连接到这个仓库。
3. 让 AI 按照本项目执行。
4. 发布任何内容前，先检查文件。

## 核心想法

一个仓库可以成为复习的操作系统。

它应该包含：

- 来源规则；
- 题目模型；
- 写作素材卡；
- 每日复盘模板；
- 控制台；
- 给未来 AI Agent 的接手规则。
