Repo as Review OS

A deployable review system built with GitHub and Markdown.

Problem

Study context is often scattered across notes, screenshots, chats, and folders.

Solution

Use a repository as the review operating system. Keep sources traceable, turn mistakes into reusable models, and give future tools a clear map before they edit anything.

What it contains

  • source rules
  • problem-card templates
  • writing-material templates
  • review dashboards
  • handoff protocols
  • deployment notes

仓库即复习操作系统

一个基于 GitHub 和 Markdown 的可部署复习系统。

问题

复习上下文经常散落在笔记、截图、聊天记录和文件夹里。

解决方案

把一个仓库当作复习操作系统。保留可追踪来源，把错题转化为可复用模型，并让后续工具在编辑前拥有清晰地图。

它包含什么

  • 来源规则
  • 题目卡模板
  • 写作素材模板
  • 复习控制台
  • 接手协议
  • 部署说明
