# 网站文案

## 标题

仓库即复习操作系统

## 副标题

一个基于 GitHub 和 Markdown 的可部署复习系统。

## 简介

Repo as Review OS 把复习材料整理成结构化项目：保留可追踪来源，把错题转化为可复用模型，并为后续工具提供清晰地图。

## 页面板块

### 问题

复习上下文经常散落在笔记、截图、聊天记录和文件夹里。

### 解决方案

把一个仓库当作复习操作系统。

### 它包含什么

- 来源规则；
- 题目卡模板；
- 写作素材模板；
- 复习控制台；
- 接手协议；
- 部署说明。
