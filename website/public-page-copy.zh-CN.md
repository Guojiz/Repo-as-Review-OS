# 网站文案

## 标题

GitLearnOS

## 副标题

把你的 GitHub 仓库变成 AI 可维护的学习系统。

## 简介

GitLearnOS 用一个可检查、可追踪、可接手的仓库保存目标、资料来源、知识模型、知识缺口、复习计划、控制台和 AI 接手记录。

## 页面板块

### 问题

学习上下文经常散落在笔记、截图、聊天记录、文件夹和 AI 对话里。

### 解决方案

把 GitHub 仓库作为可检查的学习状态系统，让 AI 可以读取、更新、复习和持续接手。

### 它包含什么

- 来源规则；
- 学习画像模板；
- 可复用模型卡片；
- 知识缺口记录；
- 复习控制台；
- AI 接手协议；
- 部署说明。
