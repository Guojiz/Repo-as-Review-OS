# 学习与自动化政策

> 这是用户对 Agent 的长期授权边界。自然语言指令可以覆盖单次事件。

## 执行模式

- 模式：`safe-auto` / `preview` / `manual`
- 一个学习事件尽量一次写回：是 / 否
- 重复输入自动去重：是 / 否
- 完成后列出所有改动：是

## 整理

- 学科路由：自动推断 / 每次询问 / 手工
- 判科有歧义时：只有写入位置会改变才询问 / 放入该学科 inbox
- 跨学科事件：一个主要归属并链接其他学科 / 询问
- 自动关联已有目标与知识缺口：是 / 否
- 自动提炼可复用模型：仅在长期有价值时 / 从不
- 自动清除已解决问题的过时任务：是 / 否
- 无法确定关联时：询问 / 放入 inbox

## 隐私

- 默认目标仓库：private
- 保存完整对话：否 / 是
- 保存老师或他人姓名：否 / 获授权时
- 保存原始图片或文件：否 / 获授权时
- 仅保留本地的内容：

## 必须先询问

- 删除历史；
- 大范围重构；
- 覆盖原始笔记；
- 改变主要目标或本政策；
- 发布私人内容或改变可见性；
- 接触密钥或更换许可证。
