# Practice Generator

This file describes how to create a small review set from the repository.

## Input

Read:

1. recent mistakes;
2. active TODO items;
3. reusable models;
4. weak topics;
5. previous review logs.

## Output

A review set should include:

- warm-up items;
- core review items;
- one transfer item;
- a short reflection section.

## Rule

Check the relevant source files before making the set.
