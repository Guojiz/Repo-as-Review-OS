# Practice Generator / 练习生成器

This file describes how to create a small review set from the repository.  
这个文件说明如何从仓库生成一份小型复习题单。

## Input / 输入

Read:  
先读取：

1. recent mistakes / 最近错题；
2. active TODO items / 当前待补项；
3. reusable models / 可复用模型；
4. weak topics / 薄弱主题；
5. previous review logs / 过去复盘记录。

## Output / 输出

A review set should include:  
一份复习题单应包括：

- warm-up items / 热身题；
- core review items / 核心复习题；
- one transfer item / 一个迁移题；
- a short reflection section / 简短复盘区。

## Rule / 规则

Check the relevant source files before making the set.  
生成题单前，先检查相关来源文件。
