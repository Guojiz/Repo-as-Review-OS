# Content Integrity Protocol

Use this protocol to avoid confusing notes with sources.

## Labels

- `source`: complete material
- `summary`: compressed explanation
- `index`: navigation only
- `model`: reusable pattern
- `todo`: incomplete item

## Checks

Before using a file, ask:

1. Is it complete?
2. Is it original or only a summary?
3. Does it link to the source?
4. Is it safe to publish?
5. What is the next action?

## Rule

If the source is missing, mark the item as `todo`.
