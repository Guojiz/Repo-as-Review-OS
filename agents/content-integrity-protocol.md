# Content Integrity Protocol / 内容完整性协议

Use this protocol to avoid confusing notes with sources.  
使用这个协议，避免把笔记和来源混在一起。

## Labels / 标签

- `source`: complete material / 完整材料
- `summary`: compressed explanation / 压缩说明
- `index`: navigation only / 仅用于导航
- `model`: reusable pattern / 可复用模型
- `todo`: incomplete item / 待补内容

## Checks / 检查

Before using a file, ask:  
使用文件前，先问：

1. Is it complete? / 它完整吗？
2. Is it original or only a summary? / 它是原始材料，还是只是摘要？
3. Does it link to the source? / 它是否链接到来源？
4. Is it safe to publish? / 它适合公开吗？
5. What is the next action? / 下一步要做什么？

## Rule / 规则

If the source is missing, mark the item as `todo`.  
如果来源缺失，把它标记为 `todo`。
