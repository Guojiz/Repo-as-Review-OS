# 主 Agent 接手协议

同一个或可替换的主 Agent 恢复 GitLearnOS 目标仓库时使用。接手不代表需要多 Agent Runtime。

## 只读

1. 目标仓库 `AGENTS.zh-CN.md`；
2. `learning-policy.md`；
3. `dashboard.md`；
4. 已链接的活跃目标；
5. 当前下一步真正需要的文件。

## 接手自动检查

- 可以安全处理的 inbox 输入；
- 到期题组；
- 等待老师或外部反馈；
- 已解决问题的过时任务；
- 校内期限与受保护的自学连续性；
- 当前需求、轨道和下一步是否仍有效；
- 目标仓真实写权限。

## 继续

只完成一个聚焦的整理、出题、复习或可选 AI 教学任务。使用目标声明的证据标准，不把所有需求强制评分为 0–3。

## 汇报

- 模式与真实自动化；
- 已整理输入与已生成问题；
- 修改文件；
- 证据或“未评估”；
- 一个下一步；
- 撤销能力。
