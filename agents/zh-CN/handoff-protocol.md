# Agent 接手协议

另一个 Agent 继续操作 GitLearnOS 目标仓库时使用。

## 只读以下内容

1. 目标仓库的 `AGENTS.zh-CN.md`；
2. `dashboard.md`；
3. 已链接的活跃目标；
4. 学习画像中相关部分；
5. 下一步行动链接的文件。

不要重新通读整个仓库。

## 验证

- 控制台刷新日期；
- 目标仓写入权限；
- 到期复习与证据分数；
- 活跃缺口与来源可用性；
- 上次下一步是否仍有效。

## 继续

只完成一个聚焦任务。保持来源完整性；作出掌握判断前先让学习者尝试；只写回长期有价值的证据。

## 汇报

- 修改文件；
- 观察到的学习证据，或“未评估”；
- 尚未解决的来源或决定；
- 一个下一步行动。
