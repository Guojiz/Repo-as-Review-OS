# Handoff Protocol / 接手协议

Use this before continuing work in a review repository.  
在继续维护复习仓库前，先读这个协议。

## Read first / 先读

1. `README.md`
2. `docs/architecture.md`
3. The relevant template in `templates/` / `templates/` 中相关模板
4. The source file being edited / 正在编辑的来源文件

## Rules / 规则

- Do not invent missing context. / 不要编造缺失上下文。
- Do not treat a summary as the original source. / 不要把摘要当作原始来源。
- Do not overwrite private material with public copy. / 不要用公开文案覆盖私有材料。
- Mark incomplete items as TODO. / 不完整内容标记为 TODO。
- Keep changes small and explain what changed. / 改动要小，并说明改了什么。

## Output after editing / 编辑后输出

Report:  
报告：

- files changed / 修改了哪些文件；
- why they changed / 为什么修改；
- what remains unfinished / 还有什么未完成。
