# Start Here

Repo as Review OS is a GitHub and AI assisted training system.

It helps a human user and an AI tool work together on a structured study repository.

## Copy this to an AI tool

```text
Read START-HERE.md, README.md, and AGENTS.md in this repository. Help me build my own Review OS. First explain the idea. Then guide me to create or choose one GitHub repository. After that, create the basic docs, agent rules, templates, and website copy.
```

## What the AI should do

1. Read this file.
2. Read `README.md`.
3. Read `AGENTS.md`.
4. Explain the system to the user.
5. Help the user create or choose one repository.
6. Build the folder structure.
7. Add templates and handoff rules.
8. Report every file created or edited.

## What the human should do

1. Create a GitHub repository.
2. Connect the AI tool to that repository.
3. Ask the AI to follow this project.
4. Review the files before publishing anything.

## Core idea

A repository can become the operating system for review.

It should contain:

- source rules;
- mistake models;
- writing material cards;
- daily review templates;
- dashboards;
- handoff rules for future AI agents.
