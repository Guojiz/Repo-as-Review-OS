# 早期公开说明

Repo as Review OS 目前是早期公开版本。

它已经可以让别人查看、复制，并交给 AI 工具试运行，但还不应该被当成最终成品。

## 已经完成

- 总运行模型；
- AI 运行指南；
- GitHub 优先的学习工作流；
- 目标模型；
- 动态规则；
- 间隔复习模型；
- 文件系统设计；
- 自动化模型；
- 一个中文轻量 demo。

## 如何试用

把这段复制给能读取 GitHub 的 AI 工具：

```text
请阅读 https://github.com/Guojiz/Repo-as-Review-OS
先读 OPERATING-MODEL.zh-CN.md、README.zh-CN.md、AGENT-RUNTIME.zh-CN.md，以及 examples/zh-CN/demo-zhongkao-lite 下的 demo。请解释我如何用一个 GitHub 仓库搭建自己的学习 OS。
```

## 安全提醒

不要把私人学习记录、老师文件、有版权教材、私人截图或真实考试记录放进公开仓库。

真实学习资料应该使用私有仓库。

公开仓库只适合放模板、方法和清理后的示例。

## 仍在实验

- 最佳目录结构还可能调整；
- 复习周期还可以继续调；
- 自动化提示词还需要更多示例；
- 还需要更多非考试场景 demo。

## 当前定位

它不是 Obsidian 工作流，不是 ChatGPT 专属项目，也不是多 Agent 框架。

它是一个基于 GitHub 的目标驱动型 AI 学习操作系统。
