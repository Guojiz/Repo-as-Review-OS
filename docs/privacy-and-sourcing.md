# Sourcing Rules / 来源规则

## Source rule / 来源规则

A summary is not a source. An index is not a source. A status note is not a source.  
摘要不是来源，索引不是来源，状态记录也不是来源。

Keep the original material or a clear reference to it whenever possible.  
尽量保留原始材料，或保留清楚的来源引用。

## Public rule / 公开规则

Before publishing a page or template, remove private details.  
发布页面或模板前，先移除私人细节。

## Good public content / 适合公开的内容

Good public content includes:  
适合公开的内容包括：

- folder structure / 目录结构；
- reusable templates / 可复用模板；
- method notes / 方法说明；
- fake examples / 虚构示例；
- workflow diagrams / 工作流图；
- deployment instructions / 部署说明。
