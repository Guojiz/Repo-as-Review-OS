# Sourcing Rules

## Source rule

A summary is not a source. An index is not a source. A status note is not a source.

Keep the original material or a clear reference to it whenever possible.

## Public rule

Before publishing a page or template, remove private details.

## Good public content

Good public content includes:

- folder structure;
- reusable templates;
- method notes;
- fake examples;
- workflow diagrams;
- deployment instructions.
