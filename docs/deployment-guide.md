# Deployment Guide / 部署指南

This project can be used in three ways.  
这个项目可以有三种用法。

## 1. Private study repository / 私有复习仓库

Keep personal materials private. Use the structure for notes, mistakes, drafts, and review logs.  
个人材料保持私有，用这个结构管理笔记、错题、草稿和复盘记录。

## 2. Public method repository / 公开方法仓库

Remove personal details. Keep only templates, rules, and example structures.  
移除个人信息，只保留模板、规则和示例结构。

## 3. Website page / 网站页面

Publish a short page that explains:  
发布一个简短页面，说明：

- the problem / 解决的问题；
- the repository structure / 仓库结构；
- the workflow / 工作流；
- the privacy rule / 隐私规则；
- the link to this template / 模板链接。

## Recommended path / 推荐路径

Start private, extract templates, then publish the method.  
先从私有仓库开始，抽取模板，再公开方法。
