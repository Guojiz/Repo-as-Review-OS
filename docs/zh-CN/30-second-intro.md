# 30 秒介绍

GitLearnOS 是一个基于 GitHub 的 AI 学习系统。

它把一个目标 GitHub 仓库变成 AI 可以读取、更新、复习和持续接手的学习时间线。

模板仓库提供方法、文档、skill、示例和规则。用户自己的目标仓库保存个人学习状态：

- 学习目标；
- 来源记录；
- 可复用模型；
- 知识缺口；
- 复习题单；
- 控制台；
- 下次复习日期；
- AI 接手记录。

核心想法很简单：GitHub 保存可检查的学习状态，AI 根据这些状态解释、拆解、出题、复习和接手学习过程。

基础部署不需要 GitHub Actions，不需要 API key，也不要求上传每个原始文件。

大文件或原始材料可以留在本地。GitHub 保存结构化学习记录。

一句话版本：

```text
GitLearnOS 让 AI 在一个你能查看、验证和长期接手的 GitHub 仓库里维护你的学习系统。
```
