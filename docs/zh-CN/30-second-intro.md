# 30 秒介绍

Repo as Review OS 是一套基于 GitHub 的学习仓库协议。

它把一个目标 GitHub 仓库变成 AI 可以读取、更新和持续接手的学习时间线。

模板仓库提供方法、文档、skill、示例和规则。

用户自己的目标仓库保存个人学习状态：

- 学习目标；
- 来源记录；
- 可复用模型；
- 复习题单；
- 控制台；
- 下次复习日期；
- AI 接手记录。

AI 应该先识别自己的运行环境和权限，再把结构部署到目标仓库。

基础部署不需要 GitHub Actions，不需要 API key，也不要求上传每个原始文件。

大文件或原始材料可以留在本地。GitHub 保存结构化学习记录。

一句话版本：

```text
Repo as Review OS 让 AI 在一个你能查看、验证和长期接手的 GitHub 仓库里维护你的学习系统。
```
