## 常见问题

### 这是模板仓库，还是我的学习仓库？

答：这个仓库是模板。用户要创建自己的目标仓库来保存个人学习记录。

### 它能把学习数据整理到 GitHub 吗？

答：可以。它整理的是结构化学习记录，不要求上传每个原始文件。

### 可以本地使用吗？

答：可以用于测试或本地流程。但本地访问不等于 GitHub 访问。

### 需要 GitHub Actions 或 API key 吗？

答：不需要。它们只是可选高级层。

### AI 能不能直接说自己已经部署好了？

答：只有在它确实创建或修改了目标仓库文件，并且能汇报改动时，才可以这么说。

## 警示信号

如果用户可能产生下面的理解，说明文档还不够清楚：

- 把个人学习数据写进模板仓库；
- 认为本地文件权限等于 GitHub 写入权限；
- 以为首次部署必须配置 GitHub Actions 或 API key；
- 以为每个原始文件都要上传到 GitHub；
- AI 声称拥有自己没有验证过的权限；
- 还没创建目标仓库，就直接发题目让 AI 开始写。
