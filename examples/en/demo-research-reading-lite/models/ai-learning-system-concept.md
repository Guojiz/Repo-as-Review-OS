## Concept

An AI learning system is more useful when it combines four layers:

```text
goal layer
→ source layer
→ model layer
→ review layer
```

## Recognition cue

When a tool claims to help learning, ask:

1. Does it know the user's goal?
2. Does it track sources honestly?
3. Does it extract reusable models?
4. Does it bring old knowledge back before it is forgotten?

## Why it matters

A chat alone can answer questions, but a learning system needs state.

The repository gives the AI an external structure for sources, concepts, review schedules, and progress.

## Comparison dimensions

Use this model to compare papers by asking:

- What kind of memory does the system use?
- Is the workspace editable?
- Does it schedule review?
- Does it preserve sources?
- Does it generate new tasks?
